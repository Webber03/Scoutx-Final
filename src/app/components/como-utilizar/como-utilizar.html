Como Utilizar o ScoutX

1. Navegação principal

O ScoutX possui um menu lateral (sidebar) que permite navegar entre as principais seções da aplicação:

  • Lista de Jogadores: Explore uma vasta lista de jogadores, filtre-os por posição, clube e nacionalidade entre outros.
  • Meus Favoritos: Veja os jogadores que você favoritou para acesso rápido.
  • Times: Em breve.
  • Deslogar: Encerre sua sessão.

2. Detalhes do jogador

Para ver informações detalhadas sobre um jogador:

  1. Clique em qualquer card de jogador na "Lista de Jogadores".
  2. Um modal pop-up será exibido com estatísticas, detalhes adicionais e uma foto do jogador.
  3. Dentro do modal, você pode clicar no ícone de estrela para adicionar ou remover o jogador dos seus favoritos.
  4. Clique no "X" ou fora do modal para fechá-lo.

3. Filtragem de jogadores

Na "Lista de Jogadores", utilize os filtros para refinar sua busca:

  • Selecione uma ou mais Posições para ver apenas jogadores que atuam nessas posições.
  • Escolha um Clube específico para ver seus jogadores.
  • Filtre por Nacionalidade para encontrar jogadores de um país em particular.
  • Os resultados serão atualizados automaticamente.

4. Comparação de jogadores

Compare as estatísticas de até dois jogadores:

  1. Na "Lista de Jogadores", clique na caixa de seleção "Comparar" abaixo do card do jogador.
  2. Selecione exatamente dois jogadores.
  3. Um botão "Comparar Selecionados (2/2)" aparecerá. Clique nele para abrir o modal de comparação.
  4. No modal de comparação, você verá as estatísticas lado a lado.
  5. Clique no "X" para fechar a comparação.

5. Favoritos

Acesse a seção "Favoritos" no menu lateral para visualizar e gerenciar sua lista de jogadores favoritados.

Você pode desfavoritar um jogador diretamente desta tela clicando no checkbox "Favorito".
